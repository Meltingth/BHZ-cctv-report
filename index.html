💰 งบประมาณและค่าใช้จ่ายรวม

เมื่อพิจารณายอดสุทธิจากใบเสนอราคาทั้ง 3 บริษัท พบว่า Nextgen SmartTech เสนอราคาต่ำกว่าเจ้าอื่นถึงกว่า 50% ซึ่งเกิดจากการบริหารจัดการทรัพยากรที่มีอยู่อย่างชาญฉลาด ไม่ใช่การลดคุณภาพวัสดุหลัก

สรุปประเด็นด้านราคา

  • ✔ ประหยัดงบประมาณสูงสุด: Nextgen เสนอราคาที่ 606,191 บาท ในขณะที่รายอื่นเสนอสูงถึง 1.2 - 1.5 ล้านบาท
  • ✔ ใช้โครงสร้างพื้นฐานเดิม (MA Synergy): ลดค่าใช้จ่ายเดินสาย Fiber Optic หลักโดยใช้สายเดิมที่มีอยู่
  • ✔ แยกโปรเจกต์อุปกรณ์เสริม: นิติบุคคลฯ สามารถจัดซื้อ จอ TV และ UPS ได้เองในราคาถูกกว่าผ่านผู้รับเหมา

📹 ความแม่นยำของจำนวนกล้อง

ผู้เสนอราคาที่ 1 และ 2 เสนอกล้องไม่ครบตามขอบเขตงาน (ขาดไป 18 ตัว) และใช้กล้องทรงกระบอกซึ่งไม่เหมาะกับภายในอาคาร

จุดเด่นของ Nextgen: เสนอกล้อง 116 ตัว (ครบถ้วนและครอบคลุม) และใช้กล้องทรง Dome/Turret ทั้งหมด ซึ่งสวยงามและเข้ากับโถงทางเดินคอนโดมิเนียม

⚒ การวิเคราะห์ค่าแรงติดตั้ง

ค่าแรงเฉลี่ยต่อจุดของ Nextgen ถูกที่สุด เนื่องจากใช้ประโยชน์จากรางเดินสายเดิมที่เคยทำไว้ ลดการรื้อฝ้าเพดานใหม่

ข้อตกลงสำคัญ: แม้ค่าแรงจะถูก (600 บ./จุด) แต่ Nextgen ยืนยันว่าการเดินสายจะเป็นการเปิดดาวน์ไลท์และซ่อนสายร้อยท่อใต้ฝ้า ไม่มีการเดินท่อลอยให้เสียทัศนียภาพ

⚙ กลยุทธ์การลดต้นทุน (Value Engineering) ของ Nextgen

เหตุผลสำคัญที่ทำให้งบประมาณของ Nextgen คุ้มค่าที่สุด เกิดจากการออกแบบระบบที่คำนึงถึงผลประโยชน์ของนิติบุคคลฯ โดยตัดสิ่งที่ซ้ำซ้อนออก และเพิ่มประสิทธิภาพการทำงานทดแทน

🖧

ตัด SD-Card เสริม RAID

ลดต้นทุนกล้องโดยเลือกรุ่นที่ไม่มีช่อง SD-Card แล้วชดเชยความเสี่ยงข้อมูลสูญหายด้วยระบบสำรองข้อมูล (RAID) บน NVR แทน

🔗

ต่อยอด Network เดิม

ใช้สายสัญญาณ Main Core (Fiber) เดิมที่มีอยู่ และใช้สาย CAT5E ลากเข้ากล้อง ซึ่งเพียงพอสำหรับแบนด์วิดท์กล้อง 4MP

📺

แยกงบ TV และ UPS

ไม่นำอุปกรณ์เครื่องใช้ไฟฟ้าทั่วไปมาบวกกำไร นิติบุคคลฯ สามารถซื้อจอ TV และ UPS เองได้ในราคาตลาดที่ถูกกว่า

⌚

ใช้ SLA บังคับคุณภาพ

เนื่องจากตัด SD-Card ออก จึงใช้เงื่อนไขสัญญา (SLA) บังคับให้ผู้รับเหมาต้องเข้าแก้ไขระบบอย่างรวดเร็วหากกล้องดับ

✔ ข้อเสนอแนะก่อนการลงนามสัญญา

เพื่อให้โครงการเป็นไปอย่างรัดกุม ขอเสนอให้ระบุเงื่อนไขเหล่านี้ลงในเอกสารแนบท้ายสัญญากับบริษัท Nextgen SmartTech:

  • ✓ มาตรฐานงานติดตั้ง: การติดตั้งกล้องทั้งหมดต้องร้อยสายซ่อนใต้ฝ้าเพดาน (เปิดดาวน์ไลท์) ห้ามเดินท่อลอยบริเวณโถงทางเดิน
  • ✓ ความรับผิดชอบร่วม: หากสาย Main Core เดิมมีปัญหา ทาง Nextgen ในฐานะผู้ดูแล MA ต้องแก้ไขควบคู่กับระบบกล้องใหม่โดยไม่คิดค่าใช้จ่ายเพิ่มเติม
  • ✓ การตั้งค่าความปลอดภัยข้อมูล: ผู้รับเหมาต้องดำเนินการตั้งค่าระบบ RAID ใน NVR และต้องเข้าบริการแก้ไข (Service) ตามระยะเวลา SLA อย่างเคร่งครัด
